# Quest

A Jira Clone for Project and Task Management.

## Tech Stack

- **Next.js 14+** (App Router) - Full-stack React framework
- **PostgreSQL** + **Prisma** - Database and ORM
- **NextAuth.js** - Authentication
- **Tailwind CSS** + **shadcn/ui** - Styling and components
